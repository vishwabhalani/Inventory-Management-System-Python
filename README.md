# Inventory-Management-System-Python
A simple and efficient Inventory Management System built using Python with JSON file handling for data storage. This project helps manage products, stock updates, sales tracking, and product records through a menu-driven interface.

## Features
- Add new products
- Search products by Product ID
- Update product stock
- Sell products with automatic bill generation
- Delete products
- View all available products
- Low stock warning system
- Sales report generation
- JSON-based permanent data storage

## Technologies Used
- Python
- JSON
- File Handling
- Dictionary Data Structure

## Project Highlights
- Data is stored permanently using products.json
- Automatic stock management after product sales
- Tracks sold quantity of each product
- Generates total sales reports and most sold product details
- Beginner-friendly console-based project
